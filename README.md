

## Resumo

Um projeto para aplicar conceitos de Api REST modular em Typescript com auxilio do Framework NestJs e TypeOrm

## Descrição 

Api rest destinada ao cadastro de pets (gatos até o momento)

## Instalação

```bash
$ yarn install
```

## Iniciar aplicação

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
